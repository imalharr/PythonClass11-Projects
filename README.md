## 📁 Files Overview

### 🔹 Basics

* **welcome.py** → prints a welcome message
* **two_numbers.py** → finds larger and smaller of 2 numbers
* **three_numbers.py** → finds largest and smallest of 3 numbers

---

### 🔹 Patterns (loops)

* **patterns.py** → prints:

```
*
**
***
****
*****

12345
1234
123
12
1

A
AB
ABC
ABCD
ABCDE
```

---

### 🔹 Series

* **series.py** → calculates 4 series:

1. 1 + x + x² + ...
2. 1 − x + x² − x³ + ...
3. x + x²/2 + x³/3 + ...
4. x + x²/2! + x³/3! + ...

---

### 🔹 Number Checks

* **number_types.py** → checks:

  * Perfect number
  * Armstrong number
  * Palindrome

* **prime.py** → checks if number is Prime or Composite

---

### 🔹 Math Programs

* **fibonacci.py** → prints Fibonacci series
* **gcd_lcm.py** → finds GCD and LCM

---

### 🔹 Strings

* **string_count.py** → counts:

  * vowels
  * consonants
  * uppercase
  * lowercase

* **string_ops.py** → checks palindrome + swaps case

---

### 🔹 Lists

* **list_min_max.py** → finds max and min
* **swap_list.py** → swaps even/odd positions
* **search.py** → searches element in list

---

### 🔹 Dictionary

* **students_dict.py** →

  * stores student data (roll, name, marks)
  * prints names with marks > 75

---

## ▶️ How to Run

Make sure Python is installed.

Run any file:

```
python filename.py
```

Example:

```
python fibonacci.py
```

---

## 🧠 Notes

* No advanced libraries used (except math)
